Detailed Description

Implements a SDL window and renderer object.

Author
P. Batty This module implements a simple window object, a structure containing a SDL_window with a SDL_renderer.

Creates a new SSL_Window object.

Parameters
titleThe title of the window.
xPositionThe stating X coordinate of the window.
yPositionThe stating Y coordinate of the window.
widthWidth of the window.
heightHeight of the window.
resWidthThe width of the resolution. Enter 0 for same as window width.
resHeightThe height of the resolution. Enter 0 for same as window height.
flagsThe flags for the window, a mask of any of the following: ::SDL_WINDOW_FULLSCREEN, ::SDL_WINDOW_OPENGL, ::SDL_WINDOW_HIDDEN, ::SDL_WINDOW_BORDERLESS, ::SDL_WINDOW_RESIZABLE, ::SDL_WINDOW_MAXIMIZED, ::SDL_WINDOW_MINIMIZED, ::SDL_WINDOW_INPUT_GRABBED
Returns
Pointer to a SSL_Window object on successful creation otherwise 0.

Creates a new SSL_Window object with allocated memory destroy with SSL_Window_Destroy.

If it cannot create the object it will return 0.
