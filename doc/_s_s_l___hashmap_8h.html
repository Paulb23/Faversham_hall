Detailed Description

Implements a simple hashmap structure.

Author
P. Batty This module implements a simple hashmap structure, a structure containing a key, data and a next element pointer.

Typedef Documentation

typedef struct _SSL_Hashmap_ SSL_Hashmap

SSL_Hashmap.

This object contains a unique key, data tied to the key and a next pointer to the next element in the map.
